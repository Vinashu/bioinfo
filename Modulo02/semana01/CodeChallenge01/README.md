Code Challenge: Solve the String Composition Problem.<br>
    Input: An integer k and a string Text.<br>
    Output: Compositionk(Text) (the k-mers can be provided in any order).

Sample Input:<br>
    5<br>
    CAATCCAAC<br>

Sample Output:<br>
    CAATC<br>
    AATCC<br>
    ATCCA<br>
    TCCAA<br>
    CCAAC    <br>
