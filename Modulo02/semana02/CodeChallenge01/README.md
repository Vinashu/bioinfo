Code Challenge: Solve the Eulerian Cycle Problem.<br>
     Input: The adjacency list of an Eulerian directed graph.<br>
     Output: An Eulerian cycle in this graph.<br>

Sample Input:<br>
     0 -> 3<br>
     1 -> 0<br>
     2 -> 1,6<br>
     3 -> 2<br>
     4 -> 2<br>
     5 -> 4<br>
     6 -> 5,8<br>
     7 -> 9<br>
     8 -> 7<br>
     9 -> 6<br>

Sample Output:<br>
     6->8->7->9->6->5->4->2->1->0->3->2->6
